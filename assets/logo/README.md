# SEAM Logo Notes

## Concept

The mark is built as a routed `S` that jumps across three horizontal lanes.
That gives it two readings at once:

- a seam or stitch joining separate edges
- a serial/data path moving through discrete protocol lines

The center amber node is the "action" moment: the point where description turns
into an interaction.

## Palette

- `#0E1B1F` - graphite, for precision and technical clarity
- `#00B89C` - signal teal, for discovery, connectivity, and active links
- `#F4A11A` - action amber, for invocation and emphasis
